# Garry's Mod Egg for Pterodactyl/Pelican
 Кастомное яйцо для Garry's Mod на игровую панель Pterodactyl/Pelican с поддержкой GitHub. Для использования этого яйца соответствующие файлы представлены в последнем релизе. По своей сути является форком основного яйца от Pterodactyl с моими изменениями и дополнениями для корректной работы некоторых DLL.
